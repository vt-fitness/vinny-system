## SKILL'S EMBEDDED EXPERTISE

### Hook Engineering Principles

**3-second rule for hook:**
- Visual cue (pattern interrupt)
- Verbal hook < 12 chữ
- Text on screen (reinforce visual)
- Sound/music vibe matching content emotion

**Hook patterns by content lever:**
- Story lever → moment-specific opener ("3 năm trước mình ngồi sau ca làm...")
- Teach lever → curiosity/myth opener ("80% anh em làm sai bài này...")
- Opine lever → bold claim opener ("Cardio first morning chỉ là marketing...")
- Show lever → visual hook (no verbal needed in 0-2s)
- Entertain lever → relatable/funny opener
- Mythbust lever → universal belief flip ("'Cắt tinh bột' không phải answer...")
- React lever → quote/clip opener với commentary setup

### Algorithm Awareness 2026

**Platform-specific optimization:**

**TikTok:**
- 15-30s sweet spot
- Completion rate critical
- Hook every 5s (loop reasons)
- Comment-bait CTA

**IG Reels:**
- 30-60s preferred
- Saves > likes (depth signal)
- Hashtag mix: branded + niche + trend
- Story-led caption

**FB:**
- Longer retention OK (90s+)
- Comments matter heavily
- Hook + story-led
- Less hashtag dependency

**YouTube Shorts:**
- 30-60s
- SEO-optimized title + description
- Subscriber-gain focus
- Watch time average critical

### Caption 7-Step Structure (Vinny's signature)

```
1. Opening congratulation/acknowledgment OR strong hook
2. Before context (struggle / current state)
3. Pain point amplification
4. Decision moment / pivot point
5. Solution applied (Vinny's expertise / methodology)
6. Result + transformation (specific numbers when applicable)
7. Soft CTA / Universal lesson
```
